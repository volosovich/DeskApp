## Usage
Follow the following steps and you're good to go! Important: Npm has to be installed on your machine!
1: Clone repo
```
git clone https://github.com/volosovich/DeskApp.git
```
2: Install packages
```
npm install
```
3: Start server (includes auto refreshing)
```
npm start
```
